**Overview**

This is a simple Parking Management API built with .NET 8.
It supports parking vehicles, checking availability, and calculating parking charges.

In order to run it, it is necessary to clone the repository, have **.NET 8 SDK** installed and run CarParkingManagment.Api project.

Assumptions:
- Charge only for the full minutes parked.
- Time saved in UTC for convinience.
